Description:


This project was created by Ishan Mehta during the Machine Learning Internship conducted by Indian Institute of Technology Bhubaneswar.This project is a Flask web application designed to detect smiles in real-time video streams from the user's webcam. Leveraging computer vision and deep learning techniques, the application accesses the webcam using OpenCV, detects faces with Haarcascade, and employs a custom Convolutional Neural Network (CNN) model to predict whether the detected face is smiling or not.



Development:


All files of this project are available on the GitHub repository: Google dot com Feel free to explore, provide feedback, and contribute to its development.



Key Features:


Flask Web Application:

This project is built using Flask, micro web framework written in Python. It supports HTTPS requests to multiple pages, GET requests from XML-HTTP-request object.



Webcam Access with OpenCV:

The website utilizes OpenCV, a powerful computer vision library, to access the user's webcam and capture live video frames. OpenCV provides the necessary tools for real-time image and video processing.



Face Detection using Haarcascade:

Haarcascade is employed to perform face detection within each video frame. This pre-trained model is highly effective in identifying faces, enabling precise localization of facial features.



Custom CNN Model for Smile Prediction:

I trained a custom CNN model on a dataset comprising of smiling and non-smiling facial images. This model learns to recognize smile-associated features, enabling accurate smile detection.



Tailwind CSS Styling:

The webpage's user interface is beautifully designed using Tailwind CSS, a utility-first CSS framework. Tailwind CSS offers a wide range of pre-defined classes, streamlining the development of visually appealing and responsive web components.



Real-time Data Display using AJAX:

AJAX is implemented to make asynchronous GET requests to the server and fetch smile detection predictions in real-time. As the CNN model processes each frame, the predictions are sent to the frontend through AJAX, enabling seamless and dynamic updates to the user interface.




This Real-time Smile Detection Web Application provides a captivating user experience by leveraging state-of-the-art technologies in computer vision, deep learning, and web development. Users can enjoy watching their smiles detected in real-time as the application provides instantaneous and accurate smile predictions using the custom-trained CNN model.
